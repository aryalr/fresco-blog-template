# Cara Mengintegrasikan Template React ke dalam Proyek Next.js

Berikut adalah panduan langkah demi langkah untuk memigrasikan template React.js (seperti yang dibuat dengan Create React App) ke dalam proyek Next.js yang sudah ada.

## 1. Salin Komponen

- **Apa:** Salin semua komponen React dari direktori `src/components` template Anda.
- **Ke mana:** Tempelkan ke direktori `app/components` di proyek Next.js Anda.

**Contoh Perintah (Windows):**
```bash
xcopy ".\\Template\\React-Landing-Page-Template\\src\\components" ".\\fresco-blog\\app\\components" /E /I /Y
```

## 2. Salin Aset Publik

- **Apa:** Salin semua aset publik seperti CSS, JavaScript (JS), dan gambar (img).
- **Ke mana:** Tempelkan ke direktori `public` di proyek Next.js Anda. Anda mungkin perlu membuat folder `css`, `js`, atau `img` jika belum ada.

**Contoh Perintah (Windows):**
```bash
# Salin CSS
xcopy ".\\Template\\React-Landing-Page-Template\\public\\css" ".\\fresco-blog\\public\\css" /E /I /Y

# Salin JavaScript
xcopy ".\\Template\\React-Landing-Page-Template\\public\\js" ".\\fresco-blog\\public\\js" /E /I /Y

# Salin Gambar
xcopy ".\\Template\\React-Landing-Page-Template\\public\\img" ".\\fresco-blog\\public\\img" /E /I /Y
```

## 3. Salin File Data

- **Apa:** Jika template Anda menggunakan file data statis (misalnya, `data.json`).
- **Ke mana:** Buat direktori `app/data` di proyek Next.js Anda dan salin file tersebut ke sana.

**Contoh Perintah (Windows):**
```bash
xcopy ".\\Template\\React-Landing-Page-Template\\src\\data\\data.json" ".\\fresco-blog\\app\\data" /Y
```

## 4. Perbarui `layout.tsx` (Layout Root)

Untuk memuat CSS dan JS global dari template Anda, ubah file `fresco-blog/app/layout.tsx`.

- **Impor `Script`:** Impor komponen `Script` dari `next/script`.
- **Tambahkan Stylesheets:** Di dalam komponen `RootLayout`, tambahkan tag `<link>` untuk file CSS Anda di dalam tag `<head>`.
- **Tambahkan Skrip:** Tambahkan komponen `<Script>` untuk file JavaScript Anda di bagian bawah tag `<body>`.

**Contoh `layout.tsx` yang sudah diubah:**
```tsx
import type { Metadata } from "next";
import "./globals.css";
import Script from "next/script";

export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en">
      <head>
        <link rel="stylesheet" type="text/css" href="/css/bootstrap.css" />
        <link rel="stylesheet" type="text/css" href="/css/style.css" />
      </head>
      <body>
        {children}
        <Script type="text/javascript" src="/js/jquery.1.11.1.js" />
        <Script type="text/javascript" src="/js/bootstrap.js" />
      </body>
    </html>
  );
}
```

## 5. Perbarui `page.tsx` (Halaman Utama)

Ubah halaman utama Next.js Anda untuk merender komponen dari template.

- **Gunakan Komponen Klien:** Karena template kemungkinan besar menggunakan hook React seperti `useState` dan `useEffect`, Anda harus mendeklarasikan halaman tersebut sebagai Komponen Klien dengan menambahkan `"use client";` di baris paling atas file.
- **Impor Komponen & Data:** Impor semua komponen yang Anda salin dan file data JSON. Pastikan jalur impor sudah benar.
- **Replikasi Struktur Aplikasi:** Salin logika dari file `App.jsx` template (termasuk state, efek, dan struktur JSX) ke dalam komponen `Home` (atau `App`) di `page.tsx`.

**Contoh `page.tsx` yang sudah diubah:**
```tsx
"use client";

import React, { useState, useEffect } from "react";
import { Navigation } from "./components/navigation";
import { Header } from "./components/header";
import { Features } from "./components/features";
import { About } from "./components/about";
import { Services } from "./components/services";
import { Gallery } from "./components/gallery";
import { Testimonials } from "./components/testimonials";
import { Team } from "./components/Team";
import { Contact } from "./components/contact";
import JsonData from "./data/data.json";
import SmoothScroll from "smooth-scroll";
import "./globals.css";

const App = () => {
  const [landingPageData, setLandingPageData] = useState({});

  useEffect(() => {
    setLandingPageData(JsonData);
    new SmoothScroll('a[href*="#"]', {
      speed: 1000,
      speedAsDuration: true,
    });
  }, []);

  return (
    <div>
      <Navigation />
      <Header data={landingPageData.Header} />
      <Features data={landingPageData.Features} />
      {/* ...sisa komponen ... */}
      <Contact data={landingPageData.Contact} />
    </div>
  );
};

export default App;
```

## 6. Instal Dependensi

- Periksa file `package.json` dari template asli untuk melihat dependensi yang diperlukan (misalnya, `smooth-scroll`).
- Instal di proyek Next.js Anda menggunakan `npm install`.

**Contoh:**
```bash
npm install smooth-scroll
```

Setelah mengikuti langkah-langkah ini, template React Anda seharusnya sudah berhasil terintegrasi dan berjalan di dalam aplikasi Next.js Anda.
